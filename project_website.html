Teacher-Student Management System

Mini Project by Yash Jakhar (MCA | Roll No. 2324000495)

Project Overview

This web-based system facilitates teacher-student communication, assignment tracking, attendance management, and tutor hiring. It aims to enhance academic management and streamline administrative tasks for educators and students.

Key Features

  • Online attendance tracking
  • Marks and assignments upload
  • Faculty portfolio updates
  • Student progress reports
  • Find and hire home tutors
  • Admin panel to manage users

System Modules

  • Administrator: Manages all data, logins, and performance tracking.
  • Faculty: Updates student data, uploads assignments, manages attendance.
  • Student: Views academic progress, applies for tutors, requests leaves.

Tools & Technologies Used

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: PHP, MySQL (via XAMPP)
  • Design Tools: Notepad++, Sublime Text
  • Database: MySQL Server 2018

Design Highlights

Data Flow Diagrams (Level 0 & 1) and Entity-Relationship Diagram (ERD) were designed to represent system structure and flow.

These design components support efficient development and maintenance of the project.
